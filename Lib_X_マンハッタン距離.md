# マンハッタン距離

d((x0,y0),(x1,y1)) = |x0-x1|+|y0-y1| という距離
x,y -> u=x+y, v=x-y に45度回転させると、
d((u0,v0),(u1,v1)) = max(|u0-u1|, |v0-v1|)となり
独立して考慮することができる
